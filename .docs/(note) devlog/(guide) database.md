## CLI 명령어로 접속하는 경우
```bash
mariadb -h yorimichi.duckdns.org -u jsl3team -p
-> [비밀번호 입력](보안상 입력값 안보임)
```
-h : host (접속할 주소)  
-u : user (로그인 아이디)  
-p : password (비밀번호 입력하겠다)
